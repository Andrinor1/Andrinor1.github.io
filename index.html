Projects

A list of projects I have completed over the years.

Caterpillar

CONTROLS: WASD/Arrow keys to move, space to shoot.
This game was created by myself as well as two classmates, Sylvester Kaestner and Blake Prasch.
I did much of the coding using p5js, while Sylvester contributed art and Blake did audio as well as some coding.

A Simple RPG (Github.com)

This is a turn-based text RPG. You choose a class, and fight monsters. There is a neat little story that goes along with it.
I wrote much of the code, along with a friend Juan Flores, and the story is attributed to Silas Bruno. This was the first project we worked on together, and used object-oriented programming before my classmates understood the concept. I understand the concepts of Object-Oriented programming and can both apply them to projects as well as teach others how it works.

Dungeon Crawler (Github.com)

This is a text-based dungeon crawler in the style of games such as Zork.
The project was completed with Juan Flores and Silas Bruno. The key feature of this project was the ability to read and save files, and our method of creating a game in which anyone could create a dungeon of their own to adventure through.

D&D Character Sheet (Github.com)

A proof-of-concept project as I was learning finer details of Object-Oriented Programming.
This is able to create level 1 D&D characters using a majority of the free content available online, as well as keep track of health and roll dice. Multiple characters can be created and saved!

Board Game Sorter (Github.com)

Owning a lot of board games can make it difficult to know which you should play.
This program keeps track of your board and card games, and will give you suggestions about what you should play next! It can take into account number of players, the amount of time you have, and which board games you've played most recently!
